Uso de metatags.

Description

A tag description é utilizada por buscadores na hora de exibir o resultado da busca.

Charset

A tag charset é utilizada para definir o conjunto de caracteres de uma página, e por vezes resolve problemas como a exibição incorreta de caracteres

Robots

  • noindex: Impede a página de aparecer em um resultado de busca.
  • nofollow: dizemos ao buscador para não nos associar aos links da pagina.

Viewport

Para configurar a exibição correta da página em múltiplos tamanhos de tela.

Link

Associa um arquivo externo a página, exemplo CSS.

Style

Permite escrever o conteudo CSS diretamente no HTML.
